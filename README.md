# vue3-h5（pnpm + vue3 + pinia + ts + vant）

## vsCode插件 TypeScript Vue Plugin (Volar)

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
